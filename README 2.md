# NodeApp

Deploy:

```sh
npm install
```

Start the app in prodiction with:

```sh
npm start
```

Start the app in development with:
```sh
npm run dev
```

Load initial data to database:
```sh
npm run init-db
```

## API Documentation

Announcement list:

GET /api/anuncios

## Como arrancamos la base de datos en Linux y en Mac

./bin/mongod --dbpath ./data
